VPN

Виртуальная частная сеть удаленного доступа РУТ МИИТ необходима для организации защищенного соединения через Интернет с внутренней сетью университета.

VPN вуза позволяет подключаться к сети РУТ МИИТ из дома, что дает возможность работать удаленно, если это требуется.

При использовании VPN за пределами вуза вашему устройству динамически назначается IP-адрес МИИТа для обратного подключения к сети РУТ МИИТ.

IP-адрес МИИТа позволяет получить доступ к вычислительным ресурсам и некоторым службам вуза.

VPN обеспечивает шифрование с устройства на шлюз. Но нельзя забывать, что другие меры безопасности, такие как правильное использование пароля, исправления безопасности операционной системы, настройки брандмауэра и т. д, необходимы для защиты данных.

Виртуальная частная сеть удаленного доступа РУТ МИИТ совместима со следующими ОС:

  • Microsoft Windows 7
  • Microsoft Windows 8
  • Microsoft Windows 10

Для получения доступа к VPN, сертификата и инструкции по установке необходимо обратится к своему руководителю и отправить рапорт на почту it@miit.ru с указанием ФИО, номера телефона и почты сотрудника.

Сотруднику будет отправлен архив, содержащий:

  • инструкцию по установке «Континент_АП»
  • файл конфигурации с расширением .apcfg
  • пароль от файла конфигурации (придет на номер телефона)
  • терминальный сервер
  • установщик программы для соединения с виртуальной сетью института (Континент_АП)

Инструкция по установке и использованию «Континент_АП» доступна по ссылке.
