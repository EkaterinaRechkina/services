АСУ МИИТ

Автоматизированная Система Управления «АСУ Университет» предназначена для решения задач управления ВУЗом.

Для получения логина и пароля учетной записи в АСУ МИИТ необходимо обратится к своему руководителю и отправить рапорт на почту it@miit.ru с указанием ФИО и телефона сотрудника. Колл-центр позвонит сотруднику и сообщит его данные для входа.

Для восстановления пароля необходимо позвонить в колл-центр МИИТа +7 (495) 662-99-43 и отправить рапорт на смену пароля.

Для добавления или изменения прав пользователя в АСУ необходимо отправить рапорт на изменение прав пользователя на почту it@miit.ru.

При необходимости внесения изменений в конфигурацию АСУ отправьте рапорт на внесение изменений в АСУ на почту it@miit.ru.

Инструкции и материалы по работе с АСУ МИИТ можно найти на рабочем столе в папке Документация.

Для подключения необходимы:

  • Наличие подключения к домену для ПК пользователей, подключенных к ЛВС университета.
  • Наличие подключения по VPN для сотрудников территориально обособленных подразделений, не подключенных к ЛВС университета.
  • Наличие учетной записи в базе АСУ МИИТ.
