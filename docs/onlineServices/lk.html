Личный кабинет РУТ МИИТ

Университет предоставляет сотрудникам и учащимся данные для входа в личный кабинет РУТ МИИТ для доступа к онлайн-ресурсам и просмотру информации о себе.

В личном кабинете РУТ МИИТ можно увидеть следующую информацию:

  • Объявления и новости вуза;
  • Информацию о себе, включая реквизиты доступа к учетной записи Microsoft (см. раздел Office 365);
  • Информацию об отпуске, зарплате, стипендии;
  • Информацию о поступлении, обучении.

Вход в Личный кабинет РУТ МИИТ

Получение доступа:

  1. Абитуриенты.

    2. Абитуриенты, подающие документы очно получают данные для входа через Приемную комиссию.

    Поступающие дистанционно самостоятельно регистрируются в личном кабинете.

    Абитуриенты, подающие документы через портал Госуслуг в дальнейшем входят в личный кабинет с помощью авторизации через Госуслуги.

    Для повторного получения/восстановления реквизитов для входа в личный кабинет абитуриенту необходимо:

    • обратиться в Приемную комиссию вуза или
    • службу поддержки по адресу: support@miit.ru

  2. Студенты.

    3. В случае утери реквизитов для входа в личный кабинет студенту необходимо: обратиться в Учебный отдел или отдел кадров своего подразделения.

  3. Сотрудники.

    4. При приеме на работу для получения реквизитов для входа в личный кабинет сотруднику необходимо отбратиться в Отдел кадров своего подразделения.

    Для восстановления доступа необходимо обратиться в Отдел кадров или службу поддержки +7 (495) 662-99-43
