Учетные записи

Учетные записи необходимы для авторизации сотрудника на своем рабочем месте. Они контролируют доступ к данным, находящихся на рабочих компьютерах, дают пользователям определённые права доступа к данным.

Для создания учетной записи нового сотрудника руководителю необходимо отправить рапорт на почту it@miit.ru с указанием ФИО и телефона сотрудника. Служба поддержки позвонит сотруднику и сообщит его данные для входа.

Для восстановления пароля сотруднику необходимо обратиться в Отдел кадров своего подразделения или в Службу поддержки.

Для изменения прав сотруднику руководителю необходимо отправить рапорт на почту it@miit.ru с указанием ФИО и телефона сотрудника.
